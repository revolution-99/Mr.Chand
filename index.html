My Projects

Magic Notes

A Note Webapp

It is a note Web App using HTML, CSS, JavaScript where a person can add a note, delete the note, mark as important and search the note by title and by content.

cKart

An E-commerce Webapp

It is a fully responsive E‐commerce website using MongoDB, React.js, Express.js, Node.js and Bootstrap where a person can add products to cart and make payments, User has also access to manage dashboard, change password, change mail ID and manage orders. There is also an admin block with the feature of changing category of products and managing products.

1SAFE

A Women Safety watch

This is a prototype of a Women Safety Watch programmed in Arduino and made with Arduino UNO board, GPS and GSM module. It enables instant messaging to the contacts with exact GPS location by pressing a button only, in case of any mishap.
